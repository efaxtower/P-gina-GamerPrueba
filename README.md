# 🎮 GamerZone Beta
**Plataforma de comunidad para apasionados del gaming.**

Este proyecto es un portal interactivo diseñado para centralizar noticias, rankings y foros de discusión gamer. Desarrollado como proyecto integrador en el **CEVAC**.

## 🚀 Tecnologías Usadas
* **Frontend:** HTML5, CSS3 (Custom Properties para el modo oscuro).
* **Lógica:** JavaScript (Manipulación del DOM para estadísticas).
* **Entorno:** Linux Mint / VS Code.

---

## 🛠️ Características Principales
* **Panel de Estadísticas:** Visualización en tiempo real de usuarios y juegos.
* **Sección de Ranking:** Los mejores jugadores de la comunidad.
* **Diseño Responsive:** Adaptado para diferentes resoluciones.
